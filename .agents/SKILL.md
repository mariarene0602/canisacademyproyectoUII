# 🎯 Agente Global .agents - Proyecto Canis Academia

Este agente coordina el desarrollo de la plataforma de gestión para **Canis Academia**, especialistas en entrenamiento y comportamiento canino de alto nivel.

## Estructura del Agente

- `skills/`: Definiciones de arquitectura estética y lógica de entrenamiento.
- `scripts/`: Automatización de tareas de desarrollo.
- `resources/`: Guías para la integración con Firebase Firestore.

## Resumen de Funcionalidad

Esta base está lista para:

1. **Gestión de Alumnos (Perros)**: CRUD completo para la tabla `perros`.
2. **Seguimiento Conductual**: Control de razas, edades y niveles de obediencia.
3. **Seguridad**: Acceso administrativo para la gestión de expedientes de entrenamiento.
