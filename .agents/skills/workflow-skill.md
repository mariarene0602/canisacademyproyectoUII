# 🔄 Skill de Flujo de Trabajo

Procesos operativos para la gestión de **Canis Academia**.

## Flujos Principales

1. **Acceso de Entrenador**: Login único para el staff de la academia.
2. **Ingreso de Alumno**: Home -> Nuevo Alumno -> Datos del Perro -> Guardar.
3. **Seguimiento**: Consulta de observaciones previas antes de cada sesión.

## Control de Versiones

- Commits: `feat: add dog obedience levels`, `style: apply navy and gold theme`.
