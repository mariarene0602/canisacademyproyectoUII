# ⏱️ Skill Planificador

Cronograma de ejecución para Canis Academia.

## Tareas Críticas

1. Configuración de la colección `perros` en Firestore.
2. Diseño de la UI con el tema Azul Marino/Dorado.
3. Desarrollo del formulario con los 5 campos: Nombre, Raza, Edad, Nivel y Observaciones.
4. Implementación de un buscador por nombre de perro.

## Scripts

- `init-project.sh`, `verify-dependencies.sh`, `setup-firebase.sh`.
